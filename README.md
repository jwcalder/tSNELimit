# On the continuum limit of t-SNE for data visualization

This code is for reproducing the numerical experiments in the paper 

Calder, J., Huang, Z., Murray, R. and Pickarski, A., 2026. [On the continuum limit of t-SNE for data visualization](https://arxiv.org/abs/2604.12041). arXiv preprint arXiv:2604.12041.

The bash script `run_all.sh` runs all experiments and generates plots.
